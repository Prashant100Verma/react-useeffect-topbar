# purpose

To learn React useEffect hook 

# concepts

- lets understand side effects first,these are operations that interact with the outside world or have effects beyond components rendering.If you introduce side effects in rendering logic of a component,react would run it every time components re-renders.
Examples-
* fetching data from an API
* subscribing to events like Websocket,timers or browser events
* starting a clock
* modifying dom manually
